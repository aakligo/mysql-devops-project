# MySQL DevOps Project

## 📌 Overview
This project demonstrates a relational database system built using MySQL. It includes users, departments, and salary management.

## 🛠 Tools Used
- MySQL
- Ubuntu Linux
- Git & GitHub

## 📂 Tables
- users
- department
- salary

## 🔗 Relationships
- Users belong to departments
- Users have salary records

## 🚀 Features
- SQL table creation
- Data insertion
- JOIN queries
- Salary analytics

## 📊 Example Queries
- Employee salary report
- Average salary by department
- Top paid employees
